# Holberton School - Advanced CSS: Web Stack Portfolio

## Project Description

This project is a comprehensive study of advanced CSS techniques. focusing on building a professional, responsive, and high-performance web portfolio. 
Over the course of **33 specialized tasks**, I implemented a complete design system from scratch, focusing on modularity, scalability, and cross-browser consistency 
without the use of modern frameworks.

## Learning Objectives

By completing this project, I am able to explain and implement:

*   **Selectors & Values:** Advanced CSS selectors and property-value logic.
*   **Box Model:** The fundamental difference between block and inline styling.
*   **Layout Systems:** Building custom Grid systems using `floats` and `clear` fixes.
*   **Architecture:** Setting up CSS Variables (`:root`) for global consistency and clean code.
*   **Asset Management:** Differences between Webfonts and SVG icons.
*   **Visual Effects:** Creating background gradients and 2D/3D transformations.
*   **Animation:** Mastering `@keyframes` and transitions for interactive UI.
*   **Browser Compatibility:** Understanding vendor prefixes and CSS resets.

## How to launch the Project

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/mammadovs/holbertonschool-web_front_end.git
    ```
2.  **Navigate to the project folder:**
    ```
    cd holbertonschool-web_front_end/CSS_advanced
    ```
3.  **Open the project:**
    *   **Simple:** Open `index.html` directly in your Chrome browser.
    *   **Professional (Live Server):** If using Python, run `python3 -m http.server` and go to `http://localhost:8000`.

##  Project Milestones

The 33 tasks were organized into the following development phases:

### 1. Global Styles & Typography (Tasks 0-9)

*   Setup of custom variables for colors, font-sizes, and weights.
*   Integration of Google Fonts and removal of default link decorations.
*   Implementation of smooth scrolling transitions.

### 2. Layout & Grid Architecture (Tasks 10-21)

*   Standardizing the project with a full CSS Reset.
*   Building a responsive float-based grid system.
*   Applying universal `box-sizing: border-box`.
*   Structuring containers and section paddings for a balanced layout.

### 3. UI Components & Theming (Tasks 22-31)

*   **Dark Theme:** Implementing specific styles and fixes for dark-mode sections.
*   **Navbar:** Customizing the navigation bar, fixing its position, and styling hover states.
*   **Decorations:** Adding border-radius to images and custom quotes for testimonials using pseudo-elements.
*   **Buttons:** Styling and state management for interactive buttons.

### 4. Interactivity (Tasks 32-33)

*   Advanced integration of transitions and final UI refinements for a polished user experience.
